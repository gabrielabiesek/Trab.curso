Introdução a Programação

A matéria de Introdução a programação tinha uma carga horária de 30 horas com o professor
Luis Aldo de Carvalho. Com o objetivo de desenvolver soluções computacionais para problemas
do mundo real, analisando requisitos simples para a construção de soluções baseadas
em passos lógicos no desenvolvimento de algoritmos eficientes a partir dos elementos
básicos da programação.

  • Portugol Studio
  • JavaScript
  • Demonstrar habilidades de raciocínio lógico ao analisar e resolver problemas algorítmicos.
  • Aplicar lógica de programação na construção de soluções para desafios computacionais.
  • Interpretar e compreender enunciados de algoritmos escritos, identificando seus requisitos e a lógica necessária para resolver o problema proposto.
  • Criar algoritmos que descrevem uma sequência lógica de passos para resolver problemas específicos.
  • Identificar e diferenciar constantes e variáveis em um programa.
  • Utilizar operadores matemáticos, lógicos e de atribuição em expressões.
  • Construir expressões lógicas e aritméticas para realizar cálculos e tomar decisões.
  • Implementar comandos de entrada e saída de dados.
  • Criar estruturas condicionais para controlar o fluxo de execução de um programa com base em condicionais simples, compostas, encadeadas e em estruturas de seleção.
  • Declarar, inicializar e manipular variáveis indexadas unidimensionais e bidimensionais no processamento de conjuntos de estruturas de dados.
  • Criar procedimentos e funções para organizar blocos de código com tarefas reutilizáveis.
  • Compreender e aplicar parâmetros de valor e referência em subrotinas.
